## Как работает

```
Сайт (CTA) → /start в боте → Сегментация (4 типа аудитории)
  → Персональная подборка спикеров → Программа → Тарифы
    → Оплата → Подтверждение админом → Пост-покупка

Не купил? → Автодожим (2ч, 24ч, за 3 дня) → Обработка возражений
```

### Автодожим

Срабатывает только для тех, кто видел тарифы, не оплатил и не нажал «Не напоминай».

| Когда | Что отправляем |
|-------|----------------|
| Через 2ч после показа тарифов | Социальное доказательство + кнопка «Посмотреть тарифы» |
| Через 24ч | Мягкий дожим + кнопки «Забронировать» / «В канал» / «Не напоминай» |
| 26 марта (за 3 дня) | «Что-то останавливает?» + кнопки возражений |
| 28 марта (за 1 день) | Напоминание оплатившим: адрес, время, совет |

## Статистика

### `/stats` — воронка

Показывает полную воронку: /start → сегмент → программа → тарифы → выбор тарифа → оплата.
Включает разбивку по сегментам, тарифам, выручку, точку наибольшей потери.

### `/stuck` — застрявшие пользователи

Три группы для ручного дожима:
1. **Выбрали тариф, не оплатили** — самые горячие
2. **Смотрели тарифы, не выбрали** — тёплые
3. **Выбрали сегмент, дальше не пошли** — холодные

### Ежедневный отчёт

Автоматически отправляется в 20:00 МСК админу статистики (5419539623).

## State management

Поле `state` в БД определяет, как бот обрабатывает текстовые сообщения:

| State | Значение | Реакция на текст |
|-------|----------|-----------------|
| `new` | Только нажал /start | «Используй кнопки» |
| `segmented` | Выбрал сегмент | «Используй кнопки» |
| `awaiting_question` | Нажал «Есть вопрос» | Пересылает текст админу |
| `awaiting_payment` | Выбрал тариф | «оплатил» / фото → пересылает админу |
| `paid` | Оплата подтверждена | «Используй кнопки» |

Приоритет обработки текста:
1. `awaiting_question` → question.js
2. `awaiting_payment` + «оплатил»/фото → checkout.js
3. Всё остальное → «Используй кнопки»

## База данных

SQLite, файл `data/bot.db`. Одна таблица `users`:

| Поле | Тип | Описание |
|------|-----|----------|
| user_id | INTEGER PK | Telegram ID |
| username | TEXT | @username |
| first_name | TEXT | Имя |
| segment | TEXT | A/B/C/D |
| tariff | TEXT | standard/premium |
| paid | INTEGER | 0/1 |
| state | TEXT | new/segmented/awaiting_question/awaiting_payment/paid |
| source | TEXT | Параметр /start (site и т.д.) |
| created_at | TEXT | Первый /start |
| last_action_at | TEXT | Последнее действие |
| program_viewed_at | TEXT | Когда смотрел программу |
| tariff_shown_at | TEXT | Когда смотрел тарифы (отсчёт дожима) |
| reminded_2h | INTEGER | Дожим 2ч отправлен |
| reminded_24h | INTEGER | Дожим 24ч отправлен |
| reminded_3d | INTEGER | Дожим 3 дня отправлен |
| reminded_day_before | INTEGER | Напоминание 28 марта отправлено |
| do_not_remind | INTEGER | Отказался от напоминаний |
| blocked_bot | INTEGER | Заблокировал бота |
